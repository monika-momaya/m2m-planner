M2M PROGRAMME PLANNER v2 — Quick Start
========================================

WHAT'S NEW IN v2:
  • Logo upload — displayed in title banner
  • Files named "M2M Programme for <<Event Name>>"
  • Word doc (.docx) download with Programme + MC Script
  • Excel download (3 sheets as before)

LOCAL:
  Windows → double-click Run_App_Windows.bat
  Mac     → double-click Run_App_Mac.command
            (if blocked: right-click → Open → Open anyway)
  Opens at http://localhost:8501

STREAMLIT CLOUD (free, shareable link):
  1. GitHub → new public repo
  2. Upload: m2m_app.py + requirements.txt
  3. share.streamlit.io → New app → Deploy
  4. Share the link with your team — no installation needed!

NO PYTHON NEEDED on team members' machines when using the cloud link.
